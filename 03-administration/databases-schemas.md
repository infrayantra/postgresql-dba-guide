# Databases, Schemas & Objects

## Create Database

```sql
CREATE DATABASE app_db
  OWNER = app_owner
  ENCODING = 'UTF8'
  LC_COLLATE = 'en_US.UTF-8'
  LC_CTYPE = 'en_US.UTF-8'
  TEMPLATE = template0    -- use template0 for custom encoding/locale
  CONNECTION LIMIT = 200;
```

> Cannot change encoding or locale after creation — use `pg_dump` + recreate if needed.

## Schemas

```sql
CREATE SCHEMA IF NOT EXISTS billing AUTHORIZATION billing_admin;
CREATE SCHEMA audit;

-- Search path
SET search_path TO billing, public;
ALTER DATABASE app_db SET search_path TO app, public;
ALTER ROLE app_user SET search_path TO app, public;
```

## Object Types Quick Reference

| Object | Create | Drop |
|--------|--------|------|
| Table | `CREATE TABLE ...` | `DROP TABLE ... CASCADE` |
| Index | `CREATE INDEX ...` | `DROP INDEX ...` |
| View | `CREATE VIEW ...` | `DROP VIEW ...` |
| MV | `CREATE MATERIALIZED VIEW ...` | `DROP MATERIALIZED VIEW ...` |
| Sequence | `CREATE SEQUENCE ...` | `DROP SEQUENCE ...` |
| Function | `CREATE FUNCTION ...` | `DROP FUNCTION ...` |
| Trigger | `CREATE TRIGGER ...` | `DROP TRIGGER ...` |

## Table Design Essentials

```sql
CREATE TABLE orders (
  id          bigint GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
  customer_id bigint NOT NULL REFERENCES customers(id),
  status      text NOT NULL DEFAULT 'pending'
              CHECK (status IN ('pending','shipped','cancelled')),
  total       numeric(12,2) NOT NULL,
  created_at  timestamptz NOT NULL DEFAULT now(),
  metadata    jsonb
);

-- Useful indexes at creation
CREATE INDEX ON orders (customer_id);
CREATE INDEX ON orders (created_at DESC);
CREATE INDEX ON orders USING gin (metadata);
```

## Identity Columns vs. Serial

```sql
-- Preferred (SQL standard, PG 10+)
id bigint GENERATED BY DEFAULT AS IDENTITY

-- Legacy
id serial   -- creates sequence + default
```

## Sequences

```sql
SELECT nextval('orders_id_seq');
SELECT currval('orders_id_seq');
SELECT setval('orders_id_seq', 1000);

-- Find sequence for column
SELECT pg_get_serial_sequence('orders', 'id');
```

## Views & Materialized Views

```sql
CREATE VIEW active_orders AS
  SELECT * FROM orders WHERE status != 'cancelled';

CREATE MATERIALIZED VIEW daily_sales AS
  SELECT date_trunc('day', created_at) AS day, sum(total)
  FROM orders GROUP BY 1;

CREATE UNIQUE INDEX ON daily_sales (day);
REFRESH MATERIALIZED VIEW CONCURRENTLY daily_sales;
```

## Foreign Keys & Referential Integrity

```sql
ALTER TABLE orders
  ADD CONSTRAINT orders_customer_fk
  FOREIGN KEY (customer_id) REFERENCES customers(id)
  ON DELETE RESTRICT;

-- NOT VALID for fast add on large tables (PG 9.1+)
ALTER TABLE orders ADD CONSTRAINT ... NOT VALID;
ALTER TABLE orders VALIDATE CONSTRAINT orders_customer_fk;
```

## Rename / Move Objects

```sql
ALTER TABLE old_name RENAME TO new_name;
ALTER TABLE orders SET SCHEMA billing;
ALTER DATABASE app_db RENAME TO app_production;
```

## Drop with Dependencies

```sql
DROP TABLE parent CASCADE;  -- drops dependent FKs, views
DROP SCHEMA old_app CASCADE;
```

## Inspect Objects

```sql
\dt billing.*
\di+ orders*
\d+ orders

SELECT schemaname, tablename FROM pg_tables WHERE schemaname = 'public';
SELECT * FROM pg_depend WHERE refobjid = 'orders'::regclass;
```

## Template Databases

| DB | Purpose |
|----|---------|
| template0 | Pristine; use for CREATE DATABASE with custom options |
| template1 | Default template; objects here copied to new DBs |
| postgres | Default admin connection DB |

```sql
CREATE DATABASE new_app TEMPLATE template0;
```

## Unlogged & Temporary Tables

```sql
CREATE UNLOGGED TABLE staging (...);  -- no WAL; lost on crash
CREATE TEMP TABLE tmp_results (...);  -- session-scoped
```
